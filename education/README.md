## Template Instructions

### For Recent Graduates (<2 years experience)
- Include GPA if 3.5+
- List relevant coursework
- Include academic projects
- Mention honors, awards, scholarships
- List relevant clubs and activities

### For Experienced Professionals (5+ years)
- Keep education brief (just degree, school, year)
- Omit GPA, coursework, and activities
- Focus on advanced degrees if relevant
- Emphasize certifications over degree details

### For Career Changers
- Highlight relevant coursework or bootcamp training
- Include recent certifications
- Mention projects completed during education
- Show continuous learning

### Certifications
- List most relevant/recent first
- Include issue date (and expiry if applicable)
- Add credential ID for verification
- Focus on industry-recognized certifications
